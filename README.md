# DA36_mini2_Dionysus

**구현한 추천 시스템**
  - 와잘알 을 위한 와인 추천 => 회귀 기반
    - (와인을 잘 아는 사람. 와인의 특징을 입력하면 와인을 추천해주는 시스템.)
  - 와린이와인 추천 테스트 => 아이템 기반
    - (와인+어린이 의 합성어. 와인에 대해 잘 모르는 사람에게 와인을 추천하는 시스템.) 
  - 맛잘알 와인 추천 시스템 => 컨텐츠 기반
    - (맛을 잘 아는 사람. 음식에 어울리는 와인을 추천하는 시스템.)
